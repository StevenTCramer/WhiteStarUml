Environment Setting for Development

Setting-up Process

  1. Install Delphi 7 and Update#1
  2. Install all Required Components (check below)
  3. Install StarUML by using StarUML Windows Installer
  4. Checkout lastest StarUML source codes from CVS
  5. Install /staruml/components/plastic-components/src/PlasticComponents.dpk package.
  6. Run Delphi 7 and open /staruml/src/StarUML.dpr file.

Language/IDE

  • Delphi 7 & Update#1 (http://www.borland.com)
